# AI Meeting Summarizer

## Project Description
The AI Meeting Summarizer is an intelligent web application that transcribes audio/video meeting recordings and generates comprehensive summaries with key insights, action items, and structured reports. The platform is designed with a strong focus on trust, user authentication, scalability, and real-world usability for professionals and teams.

The application consists of a React-based frontend and a Python Flask backend, following a modular and scalable architecture suitable for real-world deployment.

## Current Features
### Frontend (React)
- User authentication and session management
- Role-based dashboards for registered users
- File upload interface for audio/video files
- Real-time transcription status monitoring
- Report download functionality (DOCX/PDF)
- History management for past summaries
- Modular and scalable React architecture

### Backend (Python Flask)
- RESTful API architecture with CORS support
- User authentication and authorization
- Audio/video file processing and transcription
- AI-powered summarization and analysis
- Report generation in multiple formats
- Background task management for long-running processes
- Structured backend using modules, routes, and utilities
- Environment-based configuration support

## Technology Stack
### Frontend: React
- Language: JavaScript (ES6+)
- Framework: React 19.x with hooks
- Testing: React Testing Library
- UI: React Icons, Web Vitals

### Backend: Python with Flask
- Language: Python 3.8+
- Framework: Flask with Flask-CORS
- AI/ML: Faster-Whisper for transcription, LLM integration
- Audio Processing: PyDub
- Document Generation: DOCX library
- Database: JSON-based user data storage
- Task Management: Custom background task system

## Project Status
Completed and functional as a full-stack web application. The system supports end-to-end meeting transcription and summarization with user management and report generation capabilities.
